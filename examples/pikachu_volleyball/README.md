# Pikachu Volleyball

經典的皮卡丘打排球遊戲重製版，使用 Ebitengine 開發。

## 遊戲特色

- 🎮 雙人對戰模式
- ⚡ 皮卡丘角色設計（簡化視覺風格）
- 🏐 物理引擎：重力、拋物線運動、碰撞反彈
- 🦘 二段跳躍系統
- 🎯 先達 15 分獲勝

## 操作說明

### 玩家 1（左側黃色皮卡丘）
- `A` / `D` - 左右移動
- `W` - 跳躍（支援二段跳）

### 玩家 2（右側紅色角色）
- `← / →` - 左右移動
- `↑` - 跳躍（支援二段跳）

### 系統快捷鍵
- `ESC` - 暫停/繼續
- `Space` - 遊戲結束後重新開始

## 遊戲規則

1. 排球落地在對方場地得 1 分
2. 中間網子會阻擋球與玩家移動
3. 角色無法穿越中線
4. 擊球角度依碰撞位置而定
5. 先達 15 分者獲勝

## 執行遊戲

```bash
cd examples/pikachu_volleyball
go run main.go
```

或從專案根目錄：

```bash
go run examples/pikachu_volleyball/main.go
```

## 技術細節

- **引擎**: Ebitengine v2
- **物理**: 自訂重力系統
- **碰撞檢測**: 圓形-矩形碰撞
- **畫面尺寸**: 800x600

## 視覺設計

使用矩形與簡單圖形繪製皮卡丘特徵：
- 黃色方形身體
- 長條耳朵（黑色尖端）
- 黑色圓形眼睛
- 紅色腮紅
- 白色排球（十字線條）

## 系統需求

- Go 1.19+
- macOS / Linux / Windows
- OpenGL 或 Metal 支援
